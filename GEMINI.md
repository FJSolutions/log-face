# Project Instructions

## General Instructions
- When generating code, follow the existing coding style and project structure.
- Ensure all new components include JSDoc comments.
- The primary language for all documentation and comments is American English.

## Coding Style
- Use 3 spaces for indentation.
- Do not use `var`, prefer `const` or `let`.
- Prefer a more functional programming style of code to a procedural or object oriented.
